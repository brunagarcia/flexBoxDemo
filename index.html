The container (white box above) have the attibutes define as: justify-content: center;
You can also use the attibutes justify-content: flex-end; to align all the elements to the right and justify-content: flex-start; to align all elements to the left.

The elements (black squares) inside the container have the same attibutes as above, making simple to just style the parent container.


The container (white box above) have the attibutes define as: align-items: baseline; which could be also center
You can also use the attibutes align-items: flex-end; to align all the elements to the right and align-items: flex-start; to align all elements to the left.

and the black boxes would have the same attributes as above.


For the container above, we added a new attribute call flex-wrap: wrap; which will wrap the elements inside the container.
You can also use the attibutes flex-wrap: nowrap; to align all the elements to the right and flex-wrap: wrap-reverse; to align all elements to the left.

We added some attributes to the elements (black squares) inside the container to make easier to visualize how the wrapping works.
